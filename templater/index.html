technology

HTML 5

HTML (Hypertext Markup Language) is used to create electronic documents (called pages) that are displayed on the World Wide Web. It describes the structure of Web pages using markup. This markup tells a web browser how to display the text, images and other forms of multimedia on a webpage.

CSS 3

CSS (Cascading Style Sheet) are used to format the layout of web page. They can be used to define text styles, table sizes, and other aspects of Web pages. It helps Web developers create a uniform look across several pages of a Web site. CSS handles the look and feel part of a web page.

JAVASCRIPT

JavaScript is a popular client-side scripting language. Its code may be placed within each page or in a external file called by requested pages. JavaScript is a prototype-based, multi-paradigm, dynamic language, supporting object-oriented, imperative, and declarative

BOOTSTREP 4

Bootstrap as a front-end web framework that was created by Twitter for faster creation of device responsive web applications. It is as a collection of CSS classes that are defined in it which can simply be used directly. It makes use of CSS, javascript, jQuery etc. in the background to create the style, effects, and actions.
